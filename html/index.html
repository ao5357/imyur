I'm Yur Link Shortener

imyur is a fast barebones link shortener. It doesn't track links in any way. But it does get Advisory provided by Google so known phishing and malware sites can't masquerade as awesome imyur.com URLs.

  • I (Brad Czerniak) coded this up in my spare time
  • The Marrow registry and Michigan education are things I care about
  • If you like it, consider contributing to dev and hosting of imyur. No pressure

I really hope you enjoy this site. If you find a bug or know of a way to make imyur better, please drop me a line. Have a great day!
